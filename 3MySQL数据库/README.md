<h1 class="article-title">MySQL 数据库</h1>

MySQL 是典型的关系型数据库。所谓关系型数据库，是指采用了关系模型来组织数据的数据库，其以行和列的形式存储数据，以便于用户理解，关系型数据库这**一系列的行和列被称为表**，**一组表组成了数据库**。

关系型数据库的需要掌握的内容一般是第三范式、事务、ACID 特性等。
